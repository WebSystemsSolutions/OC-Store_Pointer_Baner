-[RU]
Модуль предназначен для более информативного представления банеров в виде слайдшоу с активными маркерами.

Для установки модуля необходимо:
- скопировать файлы с архива в корень сайта;
- дать права на редактирование модуля: Система -> Пользователи -> Группы Пользователей;
- создать банер: Дизайн -> Точечнный баннер;
- активировать модуль: Дополнения -> Модули -> Слайдшоу c маркерами;

-[UA]
Модуль призначений для більш інформативного представлення банерів у вигляді слайд-шоу з активними маркерами.

-[ENG]
The module is designed for a more informative presentation of banners in the form of a slideshow with active markers.

To install the module you need:
- copy files from the archive to the root directory of the site;
- give permissions to edit module: System -> Users -> User Groups;
- create banner: Design -> Dot banner;
- activate the module: Add-ons -> Modules -> Slideshow with markers;
